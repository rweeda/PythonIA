Opdracht 1.2: Bits en Bytes

Doel: oefenen met binair tellen, omrekenen van decimaal naar binair, aantal mogelijke waarden bepalen en SI-maten.

Opdracht 1.2.1 Van decimaal naar binair

Schrijf de volgende getallen om naar binair:

  1. 35 =
  2. 63 =
  3. 149 =
  4. 833 =
  5. 1024 =
Toon antwoorden opdracht 1.2.1
  1. 35 (dec) = 10 0011
  2. 63 (dec)= 11 1111
  3. 149 (dec) = 1001 0101
  4. 833 (dec) = 11 0100 0001
  5. 1024 (dec) = 100 0000 0000

Opdracht 1.2.2 Van binair naar decimaal

Schrijf de volgende getallen om naar decimaal:

  1. 0000 0111(bin) =
  2. 0001 0000 (bin) =
  3. 1010 1010 (bin) =
  4. 1100 (bin) =
  5. 0110 1100 0101 (bin) =
Toon antwoorden opdracht 1.2.2
  1. 0000 0111 (bin) = 7 (dec)
  2. 0001 0000 (bin) = 16 (dec)
  3. 1010 1010 (bin) = 170 (dec)
  4. 1100 (bin) = 12 (dec)
  5. 0110 1100 0101 (bin) = 1733 (dec)

Opdracht 1.2.3 Bytes

Schrijf de waardes om naar bytes:

  1. 15 KB = bytes
  2. 1024 GB = bytes
  3. 8 TB = bytes
  4. 63 MB = bytes
  5. 32 bits = bytes
Toon antwoorden opdracht 1.2.3
  1. 15 KB = 15.000 bytes
  2. 1024 GB = 1.024.000.000.000 bytes
  3. 8 TB = 8.000.000.000.000 bytes
  4. 63 MB = 63.000.000 bytes
  5. 32 bits = 4 bytes
