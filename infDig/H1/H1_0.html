1: Bits en Bytes

In dit onderwerp krijg je inzicht in hoe informatie digitaal wordt gecodeerd. Digitale informatie wordt opgeslagen met behulp van binaire getallen: nullen (0) en enen (1). Deze getallen vormen de basis van alle digitale communicatie en opslag.

Een bit is het kleinste stukje data en heeft slechts twee mogelijke waarden: 0 of 1.

Leerdoelen:

  • Ik kan uitleggen dat informatie digitaal binair wordt gecodeerd.
  • Ik kan tellen met binaire getallen.
  • Ik kan berekenen hoeveel mogelijke waarden een reeks bits kan weergeven.
  • Ik weet dat een byte 8 bits bevat.
  • Ik kan rekenen met SI-maten voor bytes.
  • Ik kan binaire getallen omzetten naar decimale waarden.
  • Ik kan decimale getallen omzetten naar binair.
