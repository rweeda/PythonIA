Opdracht 3.6 Fout opsporen door waardes te bekijken

Na regel 5:getal1 = getal1 - 1 heeft getal1 de waarde 0.
Op regel 7:getal3 = getal2 / getal1 wil je dus door 0 delen. Dat kan niet. Het programma levert daarom de foutmelding op: "ZeroDivisionError: division by zero" 
getal1 = 5
getal2 = 4
getal3 = 2
getal1 = getal3 - 1
getal1 = getal1 - 1 #hier wordt getal1 gelijk aan 0
print(getal1)
getal3 = getal2 / getal1 #door 0 delen kan niet
print(getal3)

Opdracht 3.7 Fout opsporen door types te bekijken

a en b zijn ints (gehele getallen). Python telt twee ints bij elkaar op, dus 1+4 wordt 5.
c en d zijn strings (teksten). Python plakt twee strings aan elkaar, dus "1"+"14" wordt 14.

Gebruik de volgende code:

a = 1
print( "a:", a, "type van a:", type(a) )
b = 4
c = "1"
print( "c:", c, "type van c:", type(c) )
d = "4"
print( a + b )
print( c + d )

Afsluitende Opdracht 3.5 Kortingsrad 

import random

print("Geef je aankoopbedrag op:")
invoer = input()
bedrag = float (invoer)

kortings_percentage = random.randint(0,25)
print("Je krijgt " + str(kortings_percentage) + "% korting op" + str(bedrag) + "euro.")

nieuw_bedrag = bedrag * (1 - kortings_percentage/100)
nieuw_bedrag = round (nieuw_bedrag, 2)
print("Het nieuwe bedrag is " + str(nieuw_bedrag) + "euro")
