Verhaal

Het verhaal geeft betekenis aan wat de speler doet. Zelfs een simpel spel heeft vaak een context of een reden waarom de speler iets doet. Een goed verhaal kan een spel veel meeslepender maken.

Voorbeeld: Tetris

Tetris heeft bijna geen verhaal. Toch kun je het zien als een strijd tegen de chaos: de speler probeert orde te scheppen voordat het scherm vol raakt. Veel moderne games voegen wel echte verhalen toe om spelers betrokken te houden.

Opdracht: Backcover Verhaal

Doel: Geef jouw game betekenis door een verhaal te bedenken.

Opdracht:

  1. Bedenk wie de hoofdpersoon is en wat het doel van de game is.
  2. Schrijf een korte ‘backcover-tekst’, alsof jouw game te koop is in een winkel.
  3. Gebruik max. 100 woorden en probeer de speler nieuwsgierig te maken.
  4. Lees je tekst voor aan een klasgenoot en vraag: zou jij dit spel willen spelen?
