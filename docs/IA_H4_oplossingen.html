Opdracht 4.15 Geneste condities

Geen uitwerking beschikbaar
  • Op regel 1 krijgt de variabele leeftijd de waarde 21.
  • De if op regel 2 levert False op, dus wordt doorgegaan naar regel 4;
  • De else op regel 4 wordt uitgevoerd omdat de bijbehorende if op regel 2 False was, er wordt doorgegaan naar regel 5;
  • De if op regel 5 levert False op, dus wordt er doorgegaan naar regel 7;
  • De else op regel 7 wordt uitgevoerd omdat de bijbehorende if op regel 5 False was, er wordt doorgegaan naar regel 8;
  • De if op regel 8 levert True op, dus wordt het bijbehorende blok uitgevoerd: print( "jong" );
  • Omdat de if op regel 8 True was, wordt de else op regel 10 niet uitgevoerd. De coderegels 11 tot en met 14 horen bij die else en worden daarom overgeslagen. Er wordt naar het einde van de constructie op regel 15 gesprongen;
  • De volgende regel code na de if-else wordt uitgevoerd, dus print("Klaar!").

vorige_gewicht = float( invoer )

print( "Wat is je huidige gewicht?" )
invoer = input()
huidige_gewicht = float( invoer )



if vorige_gewicht == huidige_gewicht:
	print("Je weegt nog precies hetzelfde als de vorige keer.")

else:
	if huidige_gewicht > vorige_gewicht:
		verschil = (huidige_gewicht - vorige_gewicht)*1000
