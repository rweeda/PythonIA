Onderwerp 11 - Python Dictionaries: Uitwerkingen

Opdracht 11.1 Dictionary met een trainingsschema maken en afdrukken

trainingsschema = {"maandag":"hardlopen", "woensdag":"fitness", "zaterdag":"voetbal"}
print(trainingsschema)                                  # print de dictionary

training_zat = trainingsschema["zaterdag"]              # haalt de waarde van de sleutel "zaterdag", namelijk voetbal
print( training_zat )

training_dinsdag = trainingsschema["dinsdag"]           # haalt de waarde van de sleutel "dinsdag", maar deze bestaat niet
print( training_dinsdag )                               # 'dinsdag' is geen sleutel, dus geeft dit een foutmelding KeyError

Opdracht 11.2 Waarde met get() ophalen

trainingsschema = {"maandag":"hardlopen", "woensdag":"fitness", "zaterdag":"voetbal"}

#opdracht a
training_wo = trainingsschema.get("woensdag")  # haalt de waarde van de sleutel "woensdag", namelijk fitness
if training_wo is None: # er is geen training
    print("Geen training!") # print dan "Geen training!"
else:
    print( training_wo )

#opdracht b
training_zondag = trainingsschema.get("dinsdag") # haalt de waarde van de sleutel 'zondag', maar deze bestaat niet

#opdracht b
if training_zondag is None: # er is geen training
    print("Geen training!") # print dan "Geen training!"
else:
    print( training_zondag )                # 'zondag' is geen sleutel, dus levert get() 'None' op

Opdracht 11.7 Lijsten maken van delen van het trainingsschema

trainingsschema = {"maandag":"hardlopen", "woensdag":"fitness", "zaterdag":"voetbal","zondag": "wielrennen"}

print( list(trainingsschema.keys()) )    # print een lijst van de trainingsdagen

Opdracht 11.8 Keuzes maken: lijst of dictionary

  1. Lijst: Een lijst is geschikt omdat je alleen de namen van studenten opslaat en de volgorde belangrijk kan zijn. Je hebt geen sleutel-waarde paren nodig, dus een lijst is eenvoudiger en efficiënter.
  2. Dictionary: Een dictionary is beter omdat je sleutel-waarde paren nodig hebt (naam-telefoonnummer). Dit maakt het ophalen van gegevens op basis van een sleutel (naam) veel efficiënter.
  3. Lijst: Een lijst is geschikt omdat je alleen de cijfers opslaat en je de volgorde van invoer wilt behouden om bijvoorbeeld de cijfers in volgorde van binnenkomst te verwerken.
  4. Lijst: Een lijst is geschikt als je alleen de items opslaat en de volgorde van toevoeging belangrijk is. Als je echter ook de prijzen en hoeveelheden van elk item wilt opslaan, kan een Dictionary beter zijn.
  5. Dictionary: Een dictionary is ideaal omdat je elk letter als sleutel kunt gebruiken en de frequentie als waarde. Dit maakt het gemakkelijk om de frequentie van elk letter bij te houden en op te halen.
  6. Lijst: Je maakt een lijst van de namen. Met len() bepaal je makkelijk hoeveel er in zitten.
  7. Lijst: Je maakt een lijst met de namen in de wachtrij. Degene die vooraan staat, mag als eerste in de attractie en wordt vervolgens uit de lijst verwijderd. Een lijst is geschikter dan een dictionary voor deze taak, omdat een wachtrij een geordende structuur heeft waarin het principe "First In, First Out" (FIFO) geldt..
  8. Dictionary: Een dictionary is meer geschikt dan een lijst voor deze taak, omdat een dictionary directe toegang geeft tot gegevens (het cijfer) op basis van een sleutel (de naam).

Opdracht 11.9 Werken met een geneste dictionary

dieren = {
    "leeuw": {"soort": "zoogdier", "leefgebied": "Afrika", "leeftijd": 14},
    "pinguïn": {"soort": "vogel", "leefgebied": "Antarctica", "leeftijd": 20},
    "python": {"soort": "reptiel", "leefgebied": "jungle", "leeftijd": 25}
}

#a: Print alle informatie over de Pinguïn
print("Alle informatie over de Pinguïn:", dieren["Pinguïn"])

#b: Print het leefgebied van de Pinguïn 
print("De leefgebied van de Pinguïn is:", dieren["Pinguïn"]["leefgebied"])

#c: Voegt een Olifant toe
dieren["Olifant"] = {"soort": "Zoogdier", "leefgebied": "Afrika", "leeftijd": 70}

#d: Verandert de leeftijd van de Leeuw
dieren["Leeuw"]["leeftijd"] = 16

#e: Drukt de informatie van alle dieren (netjes) af
print("Alle dieren:")
for dier, eigenschappen in dieren.items():
    print(dier, "is van het soort", eigenschappen['soort'], "leeft in", eigenschappen['leefgebied'], "en is", eigenschappen['leeftijd'], "jaar oud.")


#f: Doorloopt de dictionary met een for-loop en vindt het oudste dier
oudste_dier = ""
max_leeftijd = 0
for dier, eigenschappen in dieren.items():
    if eigenschappen["leeftijd"] > max_leeftijd:
        max_leeftijd = eigenschappen["leeftijd"]
        oudste_dier = dier


print("Het oudste dier is " + oudste_dier + " met " + str(max_leeftijd) + " jaar.")

Afsluitende opdracht 11.1 Aantal voldoendes

cijfers = {
    "wiskunde": 7.5,
    "Nederlands": 6.8,
    "Engels": 7.2,
    "biologie": 5.4,
    "geschiedenis": 7.0,
    "scheikunde": 4.9,
    "informatica": 8.3
}

aantal_onvoldoendes = 0

for cijfer in cijfers.values():
    if cijfer < 5.5:
        aantal_onvoldoendes += 1

print("Aantal onvoldoendes:", aantal_onvoldoendes)

Afsluitende opdracht 11.2 Bouwmaterialen

bouwmaterialen = {
    "Speler1": {"hout": 150, "steen": 90, "metaal": 50},
    "Speler2": {"hout": 200, "steen": 75, "metaal": 30},
    "Speler3": {"hout": 50, "steen": 5, "metaal": 10}
}

#a totaal aan bouwmaterialen per speler
for speler, materialen in bouwmaterialen.items():
    speler_totaal = 0  # Start met 0
    for hoeveelheid in materialen.values():  # Loop door de waarden (hoeveelheden)
        speler_totaal += hoeveelheid  # Tel de waarde op bij totaal
    print(speler, "heeft in totaal", speler_totaal, "aan bouwmaterialen.")

#b totaal aan bouwmaterialen voor alle spelers bij elkaar
totaal = 0
for speler, materialen in bouwmaterialen.items():
    for hoeveelheid in materialen.values():  # Loop door de waarden (hoeveelheden)
        totaal += hoeveelheid  # Tel de waarde op bij totaal
print("In totaal is er", totaal, "aan bouwmaterialen.")
