Opdracht 11.1 Dictionary met een trainingsschema maken en afdrukken

trainingsschema = {"maandag":"hardlopen", "woensdag":"fitness", "zaterdag":"voetbal"}
print(trainingsschema)                                  # print de dictionary

training_zat = trainingsschema["zaterdag"]              # haalt de waarde van de sleutel "zaterdag", namelijk voetbal
print( training_zat )

training_dinsdag = trainingsschema["dinsdag"]           # haalt de waarde van de sleutel "dinsdag", maar deze bestaat niet
print( training_dinsdag )                               # 'dinsdag' is geen sleutel, dus geeft dit een foutmelding KeyError

Opdracht 11.2 Waarde met get() ophalen

trainingsschema = {"maandag":"hardlopen", "woensdag":"fitness", "zaterdag":"voetbal"}

#opdracht a

if training_wo is None: # er is geen training
    print("Geen training!") # print dan "Geen training!"
else:
    print( training_wo )

#opdracht b
training_zondag = trainingsschema.get("dinsdag") # haalt de waarde van de sleutel 'zondag', maar deze bestaat niet

#opdracht b
if training_zondag is None: # er is geen training
    print("Geen training!") # print dan "Geen training!"
else:
    print( training_zondag )                # 'zondag' is geen sleutel, dus levert get() 'None' op

Opdracht 11.8 Keuzes maken: lijst of dictionary

  1. Lijst: Een lijst is geschikt omdat je alleen de namen van studenten opslaat en de volgorde belangrijk kan zijn. Je hebt geen sleutelwaarde-paren nodig, dus een lijst is eenvoudiger en efficiënter.
  2. Dictionary: Een dictionary is beter omdat je sleutelwaarde-paren nodig hebt (naam-telefoonnummer). Dit maakt het ophalen van gegevens op basis van een sleutel (naam) veel efficiënter.
  3. Lijst: Een lijst is geschikt omdat je alleen de cijfers opslaat en je de volgorde van invoer wilt behouden om bijvoorbeeld de cijfers in volgorde van binnenkomst te verwerken.
  4. Lijst: Een lijst is geschikt als je alleen de items opslaat en de volgorde van toevoeging belangrijk is. Als je echter ook de prijzen en hoeveelheden van elk item wilt opslaan, kan een dictionary beter zijn.
  5. Dictionary: Een dictionary is ideaal omdat je elke letter als sleutel kunt gebruiken en de frequentie als waarde. Dit maakt het gemakkelijk om de frequentie van elke letter bij te houden en op te halen.
  6. Lijst: Je maakt een lijst van de namen. Met len() bepaal je makkelijk hoeveel erin zitten.
  7. Lijst: Je maakt een lijst met de namen in de wachtrij. De naam van degene die vooraan staat, mag als eerste in de attractie en wordt vervolgens uit de lijst verwijderd. Een lijst is geschikter dan een dictionary voor deze taak, omdat een wachtrij een geordende structuur heeft waarin het principe "First In, First Out" (FIFO) geldt.
  8. Dictionary: Een dictionary is meer geschikt dan een lijst voor deze taak, omdat een dictionary directe toegang geeft tot gegevens (het cijfer) op basis van een sleutel (de naam).

Afsluitende opdracht 11.2 Bouwmaterialen

bouwmaterialen = {
    "Speler1": {"hout": 150, "steen": 90, "metaal": 50},
    "Speler2": {"hout": 200, "steen": 75, "metaal": 30},
    "Speler3": {"hout": 50, "steen": 5, "metaal": 10}
}

#a totaal aan bouwmaterialen per speler
for speler, materialen in bouwmaterialen.items():
    speler_totaal = 0  # Start met 0
    for hoeveelheid in materialen.values():  # Loop door de waarden (hoeveelheden)
        speler_totaal += hoeveelheid  # Tel de waarde op bij totaal
    print(speler, "heeft in totaal", speler_totaal, "aan bouwmaterialen.")

#b totaal aan bouwmaterialen voor alle spelers bij elkaar
totaal = 0
for speler, materialen in bouwmaterialen.items():
    for hoeveelheid in materialen.values():  # Loop door de waarden (hoeveelheden)
        totaal += hoeveelheid  # Tel de waarde op bij totaal
print("In totaal zijn er", totaal, "bouwmaterialen.")

Afsluitende opdracht 11.3 Geblesseerd

import json

# De dictionary
trainingsschema = {
    "maandag": "hardlopen",
    "zaterdag": "voetbal",
    "zondag": "wielrennen"
}

# a. Wegschrijven naar een JSON-bestand
uitvoer_bestand = open('trainingsschema.json', 'w') # open het bestand met schrijfrechten ('w')
json.dump(trainingsschema, uitvoer_bestand) 

# b: Lees het bestand in
invoer_bestand = open('trainingsschema.json', 'r') # open het bestand met leesrechten ('r')
trainingsschema = json.load(invoer_bestand) 

# c: Print de originele inhoud
print("Origineel trainingsschema:")
print(trainingsschema)

# d: Verwijder zaterdag (voetbaltraining)
del trainingsschema["zaterdag"]

# e: Verander hardlopen op maandag naar wandelen
trainingsschema["maandag"] = "wandelen"

# f: Voeg donderdag - zwemmen toe
trainingsschema["donderdag"] = "zwemmen"

# g: Schrijf het aangepaste trainingsschema terug
json.dump(trainingsschema, uitvoer_bestand) 

# h: Lees het bestand opnieuw in en print het nieuwe schema
nieuw_schema = open('trainingsschema.json', 'r')
nieuw_schema = json.load(nieuw_schema)

print("\nAangepast trainingsschema:")
print(nieuw_schema)
