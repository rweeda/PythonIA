Quiz over de logische laag en de toepassingenlaag

Vraag 1

Wat is de belangrijkste taak van het besturingssysteem in de logische laag?

  1. Verbinding maken met internet om updates te downloaden
  2. Verbinding maken met andere apps via Bluetooth
  3. Communicatie regelen tussen apps en de hardware
  4. Het maken van back-ups van gebruikersgegevens
Antwoord: C

Vraag 2

Welke dienst van de logische laag zorgt dat gebruikers kunnen typen en klikken?

  1. Toegang tot opslag
  2. Invoer verwerken
  3. Uitvoer regelen
  4. Geheugenbeheer
Antwoord: B

Vraag 3

Welke van de onderstaande onderdelen hoort niet bij de logische laag?

  1. Drivers
  2. Besturingssysteem
  3. Processor
  4. Systeemdiensten
Antwoord: C

Vraag 4

Wat is een kenmerk van een netwerktoepassing?

  1. Het draait volledig zonder internetverbinding
  2. Het is altijd vooraf geïnstalleerd op je computer
  3. Het draait deels of volledig op een externe server
  4. Je hebt een cd-rom nodig om het te starten
Antwoord: C

Vraag 5

Wat regelt het besturingssysteem als je een foto opent in een app?

  1. Alleen de internetverbinding
  2. De besturing van het toetsenbord
  3. Opslag, geheugenverdeling en toegang tot de afbeelding
  4. De snelheid van de fotobewerkingsapp
Antwoord: C

Vraag 6

Waardoor weet een app hoe hij met hardware moet communiceren?

  1. Via drivers en systeemsoftware
  2. Via het fysieke geheugen
  3. Via directe toegang tot de processor
  4. Via een los apparaat
Antwoord: A

Vraag 7

Welke van de onderstaande programma’s is een voorbeeld van een lokaal programma?

  1. Google Docs
  2. Netflix
  3. Magister
  4. Microsoft Word (offline)
Antwoord: D

Vraag 8

Welke taak hoort bij de toepassingenlaag in het drielagenmodel?

  1. Beheren van werkgeheugen
  2. Gebruiken van een app zoals Word of Instagram
  3. Verwerken van invoer via het toetsenbord
  4. Aansturen van hardware
Antwoord: B
